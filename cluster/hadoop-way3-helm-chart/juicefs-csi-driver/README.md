# 安装
```shell
helm install juicefs-csi-driver .  -n bigdata-local
helm uninstall  juicefs-csi-driver  -n bigdata-local              
```

# 测试
```shell
kubectl apply -f  test.yaml  -n  bigdata-local 
 kubectl delete -f  test.yaml  -n  bigdata-local 
```
成功后，可以看到启动的app会同时启动一个pod。
